# music-association-analysis

Анализ музыкальных предпочтений пользователей Last.fm с использованием алгоритмов ассоциативных правил.

## Описание проекта

Проект анализирует данные о музыкальных предпочтениях пользователей Last.fm:
- Поиск частых комбинаций исполнителей
- Анализ ассоциативных правил между артистами
- Сравнение предпочтений по странам и полу
- Визуализация результатов

## Структура проекта

- `data/` - исходные данные
- `notebooks/` - Jupyter notebook с анализом
